# Nalla Nudi

Kannada Educational Learning App built using Kotlin and Jetpack Compose.

## Features
- Kannada word learning
- Categories:
  - Maths
  - Science
  - Social
  - Technology
  - Education
- Search functionality
- Save favorite words
- English pronunciation using TextToSpeech
- Attractive UI

## Tech Stack
- Kotlin
- Jetpack Compose
- Android Studio
